Diggin PEAR channel

Registering the channel: 
pear channel-discover pear.diggin.musicrider.com
Listing available packages: 
pear remote-list -c diggin
Installing a package: 
pear install diggin/package_name
Installing a specific version/stability: 
pear install diggin/package_name-1.0.0
pear install diggin/package_name-beta
Receiving updates via a feed: 
http://pear.diggin.musicrider.com/feed.xml
